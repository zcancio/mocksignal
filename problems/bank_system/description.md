# Bank System

A progressive banking-system simulation. Implement the `Simulation` class in
`solution.py`, one level at a time — each level's spec is in its own tab and
unlocks once the previous level's tests pass.

Operations take a `timestamp` (a non-decreasing integer) as their first
argument. See the level tabs for the exact behaviour each method must have.

_Imported from the [LibreSignal](https://github.com/EricZheng0404/LibreSignal)
practice repo._
