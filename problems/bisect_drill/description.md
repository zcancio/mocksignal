# Bisect Drill

Six small functions, ~2&ndash;3 minutes each. Every function operates on a
**sorted** list; implement each one using `bisect`. No loops, no
`if x in arr` &mdash; the whole point is to drill the `bisect` syntax until
it's automatic.

The test file is the spec. Each function has its own `Test...` class, so
you'll see the six classes in the Unit Tests pane as you work; you can
implement them in any order.

If you get stuck, scroll to the bottom of `solution.py` &mdash; there's a
cheat sheet of the common `bisect` patterns left there on purpose.

_Open `tests.py` from the Files panel for the exact expected behaviour._
