# In-Memory Database

A progressive in-memory database of records, fields, and values. Implement the
`InMemoryDatabase` class in `solution.py`, one level at a time — basic field
operations, prefix scans, timestamped writes with TTL, and backup/restore.

Each level's spec is in its own tab and unlocks once the previous level's
tests pass.

_Imported from the [LibreSignal](https://github.com/EricZheng0404/LibreSignal)
practice repo._
