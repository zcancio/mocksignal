# Load Balancer

Implement a load balancer that distributes requests across a pool of
backend servers. Partial credit is granted for each test passed —
press "submit" often.

**Implementation tips:** Read the question all the way through before
you start coding. Implement the levels one at a time. Don't change
existing method signatures.

## Notes

- `server_id` and `request_id` are non-empty strings.
- `capacity` is a positive integer.
- `timestamp` is a non-negative integer.
- Specific `RuntimeError` messages are not checked, only that one is raised.
